I'm Yones Maghsoudie the creator of this website

born and raised in Shiraz, Iran. From a young age, I was passionate about becoming a game developer, which led me to start coding in Python. However, my journey took a turn when I encountered JavaScript, and I fell in love with its syntax and flexibility. Since then, I’ve shifted my focus to web development and expanded my skills in multiple technologies.

Additionally, I’ve integrated two main APIs to power the content of the website:

  • The Forkify API for retrieving recipes: ( forkify-api.herokuapp.com )
  • The Cocktail DB API for alcoholic drinks data: ( thecocktaildb.com )

Whether you’re here to find your next meal inspiration or dive deep into the culinary world, I hope youfind Taste Hub as exciting as I do!

This website, Taste Hub, is my first real portfolio and serves as a comprehensive resource where users can explore recipes, search for food and alcoholic drinks, and read related articles—all in one place. This project is built with the MVC architecture, ensuring a structured and maintainable approach to web development.

It leverages various libraries and tools, including:

  • TailwindCSS for efficient styling
  • Vite as the development build tool
  • Axios for handling API requests
  • GSAP for smooth animations
  • Lenis for scroll animations
  • Swiper for beautiful carousels and sliders
  • Yup for form validation

Although I didn’t directly learn a lot from them, Jadi Mirmirani and my friend Parsa have profoundly influenced my path in programming. Their passion and approach to technology shaped my mindset and inspired me deeply. In terms of formal learning, I’ve built my JavaScript skills through Jonas Schmedtmann’s courses on Udemy and expanded my knowledge of TailwindCSS , PHP , and Laravel thanks to the guidance of Mr. Armin Hooshmand .
